# Project Memory

This repo is for building a side-by-side evaluation tool for comparing arbitrary AI agents and large models across task spaces.

Before making product, design, architecture, or eval changes in this repo, read:

- `PROJECT_BRIEF.md`
- `docs/context/agent-pm-transition-context.md`

The local `context-sync-bundle/` directory contains the user's broader Agent PM learning, portfolio, and private work context. It is intentionally gitignored. Use `docs/context/context-sync-bundle-index.md` to retrieve from it selectively; do not load it all at once.

The LiveCue PGC Skill Agent material is a separate completed portfolio project, not native SBS context. If relevant for engineering pattern inspiration only, see `docs/portfolio/livecue-skill-agent-reference.md`.

Do not use the LiveCue/PGC eval rubric, categories, or domain rules to define this project's SBS grader or task-space methodology.

For major requirement clarification or project brief updates, prefer using the local `$product-grill-steve` skill if available.

Keep `PROJECT_BRIEF.md` updated when the project direction, assumptions, architecture, evaluation methodology, or product requirements change. Treat it as the persistent project memory for future Codex/agent sessions.
